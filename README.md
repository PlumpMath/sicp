### 문제 풀어서 올리는 방법

- `연습문제 1.1`를 푼다고 하면 `src/sicp/자기아이디/ex_1_1.clj` 파일을 만들고 아래와 같은 형식으로 파일을 작성합니다.

```clojure
(ns sicp.자기아이디.ex-1-1)

(def 문제
  "두 숫자를 인자로 받아 두 값을 덧셈하여 내놓는 프로시저를 정의하라.")

;; 아래에 자유롭게 연습 문제 코드를 작성합니다.
(defn add [x y]
  (+ x y))
```

- 파일명은 _이고 `ns` 옆에 붙는 것은 - 인 것에 주의하세요.
- 서술형 문제인 경우 주석 `;; ` 뒤에 작성합니다.

## Scheme과 Clojure가 다른점

책은 Scheme으로 설명하고 있는데요. Clojure와 다른 부분들은 여기에 계속 남기겠습니다.

### 이름 붙이는 법

#### 일반 값
- Scheme
```scheme
(define size 2)
```
- Clojure
```clojure
(def size 2)
```

#### 프로시저 값
- Scheme
```scheme
(define (square x) (* x x))
```
- Clojure
```clojure
(defn square [x] (* x x))
```

### 조건문 사용

#### cond

클로저가 괄호가 하나 적다.

- Scheme
```scheme
(cond
  ((< n 0) "negative")
  ((> n 0) "positive")
  (else "zero"))
```

- Clojure
```clojure
(cond
  (< n 0) "negative"
  (> n 0) "positive"
  :else "zero")
```
